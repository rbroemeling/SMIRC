SMIRC Changelog

2011-01-05

Updated SMSTools to the newest version and needed to make a number of tweaks to SMIRC so that everything stayed working. The new version of SMSTools should provide better notifications for when the GSM modem drops off-line. Also spent a little bit more time hardening SMIRC in general against unexpected exceptions; should improve reliability a little more.

Fixed a bug in the "new user" message that was supposed to go out to unknown phone numbers that sent a message to SMIRC. It wasn't being sent out correctly, but now that has been fixed and unregistered users that send texts to SMIRC should receive it properly.

2010-09-20

SMIRC was offline for a little bit because of issues dealing with character encoding (specifically, it didn't deal well with characters that go beyond the latin-1 character set). It's back online now, and extended characters that push SMS messages beyond the latin-1 character set should work without incident now (i.e. characters like â).

At the same time I tweaked SMIRC to harden it against errors of this type; so that in the future if it encounters a similar issue it may drop the one problematic message, but the overall service won't be affected.
