SMIRC Quickstart

Joining SMIRC

If you would like to join SMIRC, just send a text message to SMIRC (at the phone number 780-729-1450) telling it what you want your user nickname to be. This nickname will be automatically prepended to all text messages that you send through SMIRC, so it is usually a good idea for it to be shorter rather than longer. User nicknames must start with a letter (A-Z) and can only contain alphanumeric characters (A-Z and 0-9). As an example, the command to join if you want your user nickname to be "Foo" is given below:

/NICK Foo
